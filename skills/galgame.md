# Galgame / 视觉小说 通用技能

## 识别当前画面状态

### 对话状态
画面特征：底部有半透明文字框，显示角色名 + 对话内容，右下角有 ▽ 小三角。
操作：点击文字框区域（y=0.5~0.75 附近）推进到下一句。
注意：如果文字还在逐字滚动（打字机效果），第一次点击让文字立刻全部显示，再点击一次才推进。

### 选项状态
画面特征：出现 2~4 个并排或竖排的文字按钮，通常在画面中央或中偏上位置，对话框消失或变小。
操作：根据攻略偏好点击对应选项按钮的中心坐标。
注意：选项通常只出现很短时间，要立刻做出判断；不要漏点或点到空白处。

### 主菜单 / 标题画面
画面特征：显示游戏标题 LOGO，有 START / CONTINUE / LOAD / EXIT 等按钮。
操作：有存档时点 CONTINUE，没有存档时点 START，绝对不要点 EXIT。

### 过场动画 / 全屏 CG
画面特征：全屏插画或场景图，没有文字框，可能有语音在播放。
操作：等待 1~2 秒；若画面静止则点击屏幕中央跳过；若有 SKIP 按钮可点击。

### 系统对话框 / 弹窗
画面特征：出现小窗口，包含确认/取消按钮（"是"/"否"/"OK"/"Cancel"）。
操作：
- 询问"是否继续/确定"→ 点"是/OK"
- 询问"是否退出游戏"→ 点"否/取消"
- 询问"是否覆盖存档"→ 点"是"

### 黑屏 / 加载中
画面特征：全黑或全白画面，可能有进度条。
操作：等待 2~3 秒，不要乱点。

## 通用操作原则

1. **优先点击，按键为辅** — 绝大多数 galgame 操作靠鼠标点击，键盘仅在确认无法点击时使用（空格/回车推进）。
2. **坐标要准** — 点击选项时瞄准文字的正中心，偏差太大会点到空白处无效。
3. **等画面稳定** — 场景切换、淡入淡出时不要操作，等画面完全显示后再点。
4. **识别文字再选** — 看清楚选项内容再决定，不要盲目点第一个。
5. **避开 UI 菜单栏** — 顶部/底部的系统按钮（SAVE/LOAD/SYSTEM/EXIT/CONFIG）在正常游玩中不需要点击。
