# Galgame / 视觉小说 通用技能

## 对话推进
当画面显示对话文字框（通常在屏幕下方，有角色名和对话内容），说明游戏在等待你推进剧情。
- 点击画面中央或文字框区域即可推进到下一句对话
- 如果对话文字右下角有一个小三角或箭头标记，表示当前这句话已经显示完毕，可以点击推进
- 如果文字还在逐字显示中（打字机效果），点击一次可以让文字立刻全部显示

## 选项选择
当画面出现多个可点击的选项按钮（通常是 2-4 个排列的文字按钮），说明需要你做出选择。
- 仔细阅读每个选项的文字内容
- 根据剧情和角色关系，选择你认为最有趣或最能推进剧情的选项
- 如果不确定，优先选择积极正面的选项
- 点击选项文字的中心位置

## 主菜单
当画面显示游戏标题和 START / CONTINUE / LOAD 等按钮时，这是主菜单。
- 如果有 CONTINUE（继续）按钮，优先点击它以继续上次的进度
- 如果没有存档，点击 START（开始）进入新游戏
- 不要点击 EXIT / QUIT / SYSTEM 等按钮

## 过场与CG
当画面是全屏的插画、场景图或黑屏过渡时，通常是过场动画。
- 等待 1-2 秒看画面是否会自动变化
- 如果画面静止不动，点击屏幕中央尝试跳过

## 系统对话框
当出现确认对话框（如 "是否覆盖存档"、"确定退出"），注意选择正确的选项。
- 通常选择 "是"/"确定"/"OK" 来继续
- 如果对话框问是否退出游戏，选择 "否"/"取消"

## 画面静止
如果连续看到相同的画面没有任何变化，说明游戏在等待你的操作。
- 尝试点击屏幕中央推进
- 如果点击无效，尝试按空格键或回车键
- 如果仍无效，可能需要寻找画面中的可交互元素
