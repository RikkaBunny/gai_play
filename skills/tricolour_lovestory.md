# 三色绘恋 (Tricolour Lovestory) 专用知识

## 游戏简介
这是一部中文视觉小说/Galgame，讲述校园恋爱故事。玩家以男主角视角体验剧情，在关键节点做出选择，影响最终结局。

## UI 布局
- 顶部菜单栏: START | CONTINUE | LOAD | BOOKMARK | EXTRA | SYSTEM | EXIT
- 对话框在画面底部，显示角色名（左侧有头像图标）和对话文字
- 对话文字右下角的小三角 ▽ 表示可以点击推进
- 底部有小图标按钮: 自动播放、快进、存档、设置等

## 操作要点
- 推进对话: 点击画面中央偏下的位置（约 y=0.5~0.7），或者直接点击对话框区域
- 不要点击顶部菜单栏的按钮（SAVE/LOAD/SYSTEM/EXIT），那些会中断游戏流程
- 选项出现时会在画面中央显示多个文字按钮，直接点击你想选择的那个

## 攻略偏好
- 优先选择更温柔、体贴的对话选项
- 主动关心女主角
- 避免逃避型或冷淡的选项
