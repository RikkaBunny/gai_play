# 三色绘恋 (Tricolour Lovestory) 专用知识

## 游戏基本信息
- 类型：中文校园恋爱视觉小说 / Galgame
- 视角：男主角第一人称
- 目标：推进剧情，在选项中做出选择，解锁各女主角路线和结局
- 女主角：三位，各有独立路线

## UI 布局说明

### 顶部菜单栏（不要点击）
`START | CONTINUE | LOAD | BOOKMARK | EXTRA | SYSTEM | EXIT`
这些按钮在正常游玩中不需要操作，点击 EXIT 会退出游戏。

### 对话界面
- 画面底部有半透明对话框
- 左侧显示角色立绘头像 + 角色名
- 文字框内显示对话内容
- 右下角 ▽ 表示这句话已显示完毕，可以点击推进
- 推进点击区域：画面中央偏下 (x=0.3~0.7, y=0.55~0.75)

### 选项界面
- 对话框消失，画面中央出现 2~3 个竖排的文字按钮
- 每个按钮都是选项，点击中心即可选择
- 选项通常位于 y=0.35~0.65 区间

### 底部工具栏（小图标）
屏幕最底部有一排小图标，从左到右大致是：
自动播放（AUTO）| 快进（SKIP）| 存档 | 读档 | 设置
正常游玩中不需要操作这些。

## 操作方法

### 推进对话
点击 (x=0.5, y=0.6) 附近，即画面中央偏下的对话框区域。

### 选择选项
看清楚每个选项文字 → 按攻略偏好点击对应按钮中心。
示例坐标（3个选项竖排时）：
- 第1个选项：约 (x=0.5, y=0.38)
- 第2个选项：约 (x=0.5, y=0.50)
- 第3个选项：约 (x=0.5, y=0.62)

### 等待场景切换
场景淡入淡出时等待 1~2 秒再操作，不要在黑屏期间点击。

## 攻略策略（偏好选项）

优先选择以下类型的选项：
- **主动关心**：询问女主角的感受、健康、烦恼
- **温柔体贴**：表示理解、陪伴、支持
- **真诚直接**：表达真实情感，不逃避
- **积极正面**：充满希望和行动力的回答

避免以下类型的选项：
- 冷淡、逃避、敷衍的回答
- 自私或只顾自己感受的选项
- 明显会惹女主角生气的选项

## 常见场景判断

| 画面状态 | 正确操作 |
|----------|----------|
| 对话框有 ▽ | 点击对话框区域推进 |
| 对话文字滚动中 | 点一次让文字立刻显示完，再点一次推进 |
| 出现选项按钮 | 读清文字，按偏好选择 |
| 全屏 CG 插画 | 等 1 秒，若静止则点击中央继续 |
| 黑屏过渡 | 等待，不要操作 |
| 主菜单 | 点 CONTINUE（或 START） |
| 弹出系统窗口 | 看清内容，不要点退出相关按钮 |
